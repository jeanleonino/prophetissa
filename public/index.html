(#) About

This is a tiny css framework (at this moment: 1.4kb uncompressed, non-minified) made just to learn some css :-) the idea is a simple framework that can be reused, with a responsive 16-column grid and these elements:

  • navbar menu
  • typography
  • modals
  • forms

Not mobile first, not better than Bootstrap Twitter, not better than what you already use, please don't use it.

PS: I'm considering to make it component-focused. But that's just a thought for now.

(#) Grid System

A 1200px-wide 16-column system grid, which makes it sixteen columns with 55px each, and 20px gutter.
